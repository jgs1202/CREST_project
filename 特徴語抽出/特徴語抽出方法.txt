・特徴語抽出方法

テキストファイルから、そのテキストを特徴づける特徴語を抽出する手順を示します。

まず前提として、この方法はユーザーがそのテキストの中で重要な意味を持つ単語を知っている必要があります。
この方法ではまずテキストファイルを単語ごとに分け、その中から重要な単語を検索し、その前後の単語を抽出しcsvファイルとして出力します。
ユーザーはそのcsvをみて特徴語を発見することが出来ます。

単語列のcsvファイルを出力する方法を以下に示します。
python3環境を使用しますので、導入されていない方はまずpython3の環境を整えてください。
単語検索に必要なパッケージは、numpy, nltkです。

*numpyのインストール

	コマンドラインで以下のコマンドを実行。
	
	>pip install numpy

*nltkのインストール
	
	コマンドライン上で以下のコマンドを実行

	>pip install -U nltk

	>ipython
	>import nltk
	>nltk.download("all")

またプログラムと同じディレクトリに'search-result'というディレクトリを作成しておく必要があります。

以上の準備で本プログラムを実行できると思います。
検索元として、n行1列のテキストデータを使用します。
拡張子はcsvで、編集前は同じディレクトリ内の'sampletxt.csv'というものを使用します。
尚検索を実行する前に本文中の ','を全て削除しておくことが望ましいです。

'sample.csv'を用意したらコマンドライン上で "> python select_word.py"を実行します。
すると検索語を要求されますので検索語を入力します。
もし検索語が'word'の場合、'word', 'Word', 'WORD'の３種のパターンでその単語をテキストデータから検索し、前後2語を含めた計5語を計200パターンずつ'search-result'内にcsvファイルとして保存します。
もう一度検索したい場合は'y', 検索を終了する場合は'n'を入力します。

検索結果からユーザーは特徴語を発見することが出来ます。
